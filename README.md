# HTML5 Project

## Description

This project is a collection of HTML5 examples and exercises from the Holberton School Web Front-End curriculum. It demonstrates various HTML5 features including semantic elements, multimedia, forms, and more.

## Features

- Semantic HTML5 elements
- Multimedia integration (audio/video)
- Interactive forms
- Canvas and SVG graphics
- Responsive design principles
- Accessibility best practices

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kedia12/holbertonschool-web_front_end.git
   ```

2. Navigate to the project directory:
   ```bash
   cd holbertonschool-web_front_end
   ```

3. Open the HTML files in your web browser or use a local server.

## Usage

Open any HTML file in your preferred web browser to view the examples. For a better development experience, consider using a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Then visit http://localhost:8000
```
